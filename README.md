## About

`IDK-IMU` 即 `I Don't Know IMU`，意思是我不知道的 IMU，取名来自经典书籍《你不知道的 JavaScript》。但在这里，我记录的是大学应该怎么学习以适应越来越严峻的就业形势。

## Start

文档生成基于 `VitePress`。

克隆：
```bash
git clone git@github.com:Hacker-C/I-Dont-Know-JS.git
```

安装yarn：
```bash
npm install -g yarn
```
下载依赖：
```bash
yarn install
```

开发：
```bash
yarn dev
```

打包：
```bash
yarn build
```
